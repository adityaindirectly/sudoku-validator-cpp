// Sudoku Validator (C++)

A command-line Sudoku validator built in C++ that checks whether a given 9×9 Sudoku board is valid.

 Features : 
- Validates rows, columns, and 3×3 subgrids
- Handles empty cells using '.'
- User-friendly input from terminal
- Efficient implementation using hash sets

Data Structures used : 
- STL (vector, unordered_set)

 How to Run : 

```bash
g++ sudokuvalidator.cpp -o sudokuvalidator
./sudokuvalidator
